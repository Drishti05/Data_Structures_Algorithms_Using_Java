Consecutive Ones
Easy

You are given a list of integers nums which contains at least one 1. Return whether all the 1s appear consecutively.

Constraints

1 ≤ n ≤ 100,000 where n is the length of nums
Example 1
Input
nums = [0, 1, 1, 1, 2, 3]
Output
true
Example 2
Input
nums = [1, 1, 0, 1, 1]
Output
false
